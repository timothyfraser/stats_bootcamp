# Module S: Calculus with Statistical Coding

Module S teaches how to use symbolic calculus in R and Python for statistical and systems engineering applications. Learn how to integrate probability density functions (PDFs) to get cumulative distribution functions (CDFs) and reliability functions (1 - CDF) for projections and simulations.
